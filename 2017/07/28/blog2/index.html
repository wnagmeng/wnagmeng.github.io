HTML_float-浮动

Css样式的float浮动属性，用于设置标签对象

div是块级

如果想两个div并排在一起，可以给div加inline-block===》

但是有空隙（两种方法取消间隙1，加注释2，div并排在一起）

这种方法比较lol

高级的方法就出现了——》float浮动

float://浮动，就是飞起来的感觉

left:左浮动

right:右浮动

文档流规则：

从左到右，从上到下；

float语法：

float : none | left |right

浮动的特点：

  • 脱离文档流，不占空间

  • 根据值-》左浮动/右浮动

  • 会影响后面的元素

  • 可以跟其他元素并排在一起

  • 浮动元素会造成父级元素高度的塌陷（塌方）

  • 浮动可以让文本环绕（会把其他元素的文本挤出去，会影响正常文档流的文字排版）

  • 父级元素宽度不够的时候，浮动元素会掉下去

  • 如果浮动元素的高度不一致的时候，父级的宽度又不够容纳的时候，后面的浮动元素往下面掉的时候，可能会被其他的高个子元素卡住。

  • 浮动元素会在碰到父元素的边缘时停下来。

  • 如果前面有一个在文档流中的元素，浮动也会停下来。

  • 浮动会改变元素的特性。给span等元素加float的时候，宽高有效。既：浮动会让元素变块级。

  • 浮动不给宽高的时候，内容撑开宽高。

  • margin-auto；虽然有宽高，但失效。因为已浮动了。

【清楚浮动】==》解决父级塌陷问题

方法：

  • right–》清除右浮动

浮动卡住：

当外盒子宽度不够，内部几个盒子一起左浮动，那么最后一个盒子会受前一个浮动盒子的高度影响。
